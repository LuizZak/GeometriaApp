# GeometriaApp

A sample Raytracer implementation using [Geometria] as a backing geometry library.

### Running:

From terminal, run:

```bash
$ make run-rel
```

or:

```bash
$ swift run
```

And wait for the project to build and a window to pop up!

You should see something along the lines of:
